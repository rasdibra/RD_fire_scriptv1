ENGLISH
RD_fire_script is a RedM script created to add a campfire system to your server, especially useful for cold weather and roleplay scenarios.

This is version v1, where the fire is created using an item (campfire).

⚙️ Features
🔥 Campfire creation using an item
🎒 Inventory integration (wood + rock)
⏳ Crafting system to start the fire
🌫️ Visual effects (fire + smoke)
🧱 Campfire object spawning
📦 Requirements

Make sure your server has the following items in the inventory:

campfire
wood
rock

Without these items, the fire cannot be started.

📥 Installation
Place the script in your resources folder
Rename the folder to:
fire_script
Add this to your server.cfg:
ensure fire_script
▶️ Usage
Use the campfire item
Select required materials (wood & rock)
Start the fire through the UI
ℹ️ Notes
This is the first version (v1)
Previous version used a command, this one uses an item
Can be expanded with temperature or survival systems
